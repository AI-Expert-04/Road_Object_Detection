# RoadObject-Detection

## Python 환경

- anaconda 4.6.11
  - python 3.6
  - numpy 1.17.3
  - matplotlib 3.1.1
  - tensorflow 2.0.1
  

## 환경 설정

1. conda 환경 생성

```
>>> conda create -n python-env python=3.6
```



2. conda 환경 활성화

```
>>> conda activate python-env
```



3. 프로젝트 경로로 이동

```
>>> cd (프로젝트 경로)
```



4. python 모듈 설치

```
>>> pip install -r requirements.txt
```
